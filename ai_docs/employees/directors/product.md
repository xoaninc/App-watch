### Feature Prioritization Framework

**RICE Score:**
- **R**each: How many users affected?
- **I**mpact: How much value per user?
- **C**onfidence: How sure are we?
- **E**ffort: Engineering weeks needed

```
RICE Score = (Reach × Impact × Confidence) / Effort
```
